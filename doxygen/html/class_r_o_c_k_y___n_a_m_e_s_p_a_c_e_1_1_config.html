Detailed Description

Config is a general-purpose container for serializable data. You store an object's members to Config, and then translate the Config to a particular format (like XML or JSON). Likewise, the object can de-serialize a Config back into member data. Config support the optional<> template for optional values.

◆ setReferrer()

void Config::setReferrer ( const std::string &  value)

Referrer is the context for resolving relative pathnames that occur in this object. For example, if the value is a filename "file.txt" and the referrer is "C:/temp/a.earth", then the full path of the file is "C:/temp/file.txt".

Calling this sets a referrer on this object and its children.
